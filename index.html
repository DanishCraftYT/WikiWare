WikiWare

WikiWare is a piece of software developed by Danish Craft to make wiki websites and applications.

Usage

to use WikiWare. first link to the WikiWare stylesheet. then add the WikiWare JavaScript script to the html file.

all of the wiki will be inside the wiki-container which will be inside the body element. inside it you can put in the wiki-page element which will contain the text for whatever you wanna write about.

wiki pages are split into sections and sub-sections. these allow you to group stuff together on your page. use wiki-section to create a section inside your page. then use wiki-subsection inside your wiki-section to create a sub-section.

Wiki Content

you can also add a content box to your wiki page. to do so. add wiki-content inside of your wiki-page element to create a content box. you can use wiki-contentsection to create a section inside of your content box. if you want it to have a title. you can add the text attribute to your wki-content element.

it's highly recommended to use wiki-sidepanel when you have multiple wiki-content's next to each other to make them appear in a row instead of a column.

Wiki Header

you can also use wiki-header to create a header for your wiki (just make sure to put it above the wiki-page element). you can use wiki-headersection to create a new section inside the header for your wiki page.

Wiki Details

you can also use wiki-details to create something like the details element in HTML. add the wiki-detailsbox elment inside the wiki-details element to create the box that is displayed when you press the button in the wiki-details element. you can add elements that you want displayed inside the details box to the wiki-detailsbox element.

"nobottomline" Attribute

wiki-section and wiki-tablecontentsection has a attribute called nobottomline which when set to "true" removes the bottom line created by their section.

editing the style of your wiki

you can edit the style of your wiki by modifiying the "WikiWare.css" stylesheet to your needs. just don't modify the ones that says "DO NOT EDIT" as it will likely lead to the wiki not rendering correctly.

HTML Elements and Attributes

a list of all HTML Elements and Attributes that are part of WikiWare.

wiki-container

this HTML Element contains the wiki.

wiki-page

this HTML Element contains the page with all the text.

wiki-content

this HTML Element creates a content box which you can add text and other things to.

wiki-content "text" Attribute

you can add the text attribute to your wiki-content element to give it a title.

wiki-contentsection

this HTML Element creates a section inside of a wiki-content element.

wiki-sidepanel

use this HTML Element if you want multiple wiki-content elements next to each other to render them in a row instead of a column.

wiki-section

this HTML Element creates a section in the wiki.

wiki-subsection

this HTML Element creates a subsection in the wiki.

wiki-header

this HTML Element creates a header for the wiki page.

wiki-headersection

this HTML Element creates a section used for the header in the wiki page.

wiki-details

this HTML Element creates a button which can be combined with a wiki-detailsbox to create something similar to the details element in HTML.

wiki-detailsbox

this HTML Element creates a box for wiki-details to either render or not render when the user presses on it.

"nobottomline" attribute

the elements wiki-section and wiki-contentsection have a attribute called "nobottomline" which when set to "true" removes the bottom line created by the section.
