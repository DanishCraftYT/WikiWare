Usage

to use WikiWare. first link to the WikiWare stylesheet. then add the WikiWare JavaScript script to the html file.

all of the wiki will be inside the wiki-container which will be inside the body element. inside it you can put in the wiki-page element which will contain the text for whatever you wanna write about.

wiki pages are split into sections and sub-sections. these allow you to group stuff together on your page. use wiki-section to create a section inside your page. then use wiki-subsection inside your wiki-section to create a sub-section.

Wiki Table Of Contents

you can also add a table of contents to your wiki page. to do so. add wiki-tablecontent inside of your wiki-page element to create a table of content box. you can use wiki-contentsection to create a section inside of your table of content box.

Wiki Info Content

you can also add a info content box to your wiki page. to do so. add wiki-content inside of your wiki-page element to create a info box. you can use wiki-contentsection to create a section inside of your wiki content box.

Wiki Header

you can also use wiki-header to create a header for your wiki (just make sure to put it above the wiki-page element). you can use wiki-headersection to create a new section inside the header for your wiki page.

"nobottomline" Attribute

wiki-section and wiki-tablecontentsection has a attribute called nobottomline which when set to "true" removes the bottom line created by their section.

editing the style of your wiki

you can edit the style of your wiki by modifiying the "WikiWare.css" stylesheet to your needs. just don't modify the ones that says "DO NOT EDIT" as it will likely lead to the wiki not rendering correctly.

HTML Elements and Attributes

a list of all HTML Elements and Attributes that are part of WikiWare.

wiki-container

this HTML Element contains the wiki.

wiki-page

this HTML Element contains the page with all the text.

wiki-tablecontent

this HTML Element creates a Table Of Contents box which you can add links inside of to link to things in this page or other pages.

wiki-infocontent

this HTML Element creates a Info box which you can add information and other things inside of.

wiki-contentsection

this HTML Element creates a section inside of a "wiki-tablecontent" or "wiki-infocontent" element.

wiki-section

this HTML Element creates a section in the wiki.

wiki-subsection

this HTML Element creates a subsection in the wiki.

wiki-header

this HTML Element creates a header for the wiki page.

wiki-headersection

this HTML Element creates a section used for the header in the wiki page.

"nobottomline" attribute

the elements "wiki-section" and "wiki-tablecontentsection" have a attribute called "nobottomline" which when set to "true" removes the bottom line created by the section.
